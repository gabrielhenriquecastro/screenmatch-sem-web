🎬 Screenmatch
Aplicação Java para busca e listagem de séries, consumindo a API do OMDB. Projeto desenvolvido durante o curso de Java na Alura.

🚀 Funcionalidades
Busca de séries pelo nome via API OMDB
Listagem de séries buscadas
Exibição de dados completos: gênero, atores, poster e sinopse
Categorização por gênero com Enum
Tradução automática da sinopse via API MyMemory


🛠️ Tecnologias
Java 21
Spring Boot
API OMDB


⚙️ Como rodar
Clone o repositório
git clone https://github.com/SEU_USUARIO/screenmatch-sem-web.git


Configure a variável de ambiente com sua chave da API OMDB
export OMDB_API_KEY=sua_chave_aqui


Execute o projeto
bash./mvnw spring-boot:run

📚 Aprendizados
Consumo de APIs externas com Java
Manipulação de listas e coleções
Lambdas e Streams com Spring Framework
